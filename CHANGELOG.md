## [0.1.0] - 2026-03-09

### Added

- CEO-driven delivery orchestration with a deterministic pipeline FSM.
- Specialized agent contracts for architecture, implementation, review, testing, and language-specific execution.
- SQLite-backed state for pipelines, stages, gates, decisions, and artifacts.
- Context packing and restore support for long-running sessions.
- GitHub delivery helpers for branch preparation, pull request creation, and repository fingerprinting.
- Stack detection for common project types with specialist routing support.
- Repository automation covering CI verification, release publishing, issue templates, PR templates, code ownership, and dependency updates.
